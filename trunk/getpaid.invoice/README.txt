getpaid.invoice is a framework to generate invoices from GetPaid system.
This product is aimed for GetPaid system integrators.

This system generates invoices for shopping carts
which satisfy the requirements of law

* Company name visible

* Company address visible

* Company VAT number visible

* VAT percent

* Price without VAT

* Price with VAT

* Payment deadline

* Terms of payment

* Reference number

Supported invoice types
-----------------------

* Plain text

* PDF
