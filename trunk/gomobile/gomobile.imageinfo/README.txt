Unified interface to access image data in different Zope image objects 
    
Return Python Imaging Library presentation of images:
    
* Skin layer images, both file system based and ZMI uploads
    
* Archetypes ATImage
    
* Resource images (static media)

Purpose
-------

This package is mainly used with Plone GoMobile project to dynamically reformat images to mobile presentation
regardless how image was inputted to system.
