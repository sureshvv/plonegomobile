Introduction
------------

This package provides mobile adaption for various Plone add on products.

Supported products
------------------

* `Feedfeeder <http://plone.org/products/feedfeeder>`_

* `Easy Tempplate <http://plone.org/products/easy-template>`_

Installation and usage
----------------------

Please refer to `Web and Mobile documentation <http://webandmobile.mfabrik.com/docs>`_.

Source code and issue tracking
-----------------------------------

The project is hosted at `Google Code project repository <http://code.google.com/p/plonegomobile>`_.
